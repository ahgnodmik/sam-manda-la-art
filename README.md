최근 업데이트 내용:

1. 엔터키 입력 가능 - 더 이상 엔터키를 막지 않음
2. 중앙 셀 스타일 변경 
   - 버건디 색상(#800020)
   - 볼드체 적용
3. 중앙 셀 플레이스홀더 기능 추가
   - 기본 텍스트 "최종목표" 표시
   - 포커스시 텍스트 사라짐
   - 입력 없이 포커스 해제시 다시 표시
4. 초기화(Clear All) 버튼의 확인 메시지 한글화
